Calories reduced

Maikel Thursday, 28 October 2021 diary

I was doing 2177 calories per day but in the last few days I've gained weight from the exact date I started doing creatine. Today it has been my last day on the loading week and I've been for 4 days weighting exactly the same unchanged 71.8 kgs.

Four days is a lot of time wasted so I'm doing something about it, I'll try to get more exercise done (what, where, when) but I'll reduce the calory intake from 2177 to 1900 per day. I haven't touch the 163 grams per day of protein but only the carbs figure.

What?

Long walks.

When?

It could be potentially after the gym (and after the nap) but only as far as it is walking. Start tomorrow.

Where?

Walk towards the beach, once there start reading.
