Learning CMS

Maikel Friday, 25 February 2022 portfolio

This is a personal project of a learning platform. I wanted to facilitate the acquisition of skills by controlling the status of each one of them, knowing where to update them, which ones do I already know, etc. While having an interface to write notes about them and test my own knowledge. As a result I built my own self-directed learning CMS on Angular 4, Node, Express and MongoDB on an Amazon EC3 Ubuntu container.

The website is fully responsive, so it adapts to mobile phones when opened from them. You're free to access the source code in github.
